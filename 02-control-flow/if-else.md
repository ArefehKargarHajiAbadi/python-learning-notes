# If / Else Statements in Python

## Syntax
```python
if condition:
    # code if condition is True
elif other_condition:
    # code if other_condition is True
else:
    # code if none are True
```
## Example
```python
x = 10
if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")
#output: "Positive"
```
